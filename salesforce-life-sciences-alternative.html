Alternative guide

HCP segmentation without a Salesforce implementation

Salesforce Life Sciences Cloud (and Health Cloud before it) is a powerful platform — when you have the licenses, the admins, and the implementation roadmap. If what's actually blocking you is "we have HCP lists in spreadsheets and we need scored, compliant segments," a platform project is the slow way there. LiteHCP Targeter is the fast way: one self-hosted PHP file that does exactly that slice.

The platform-tax problem

Spreadsheet-to-segment workflows on an enterprise platform typically require: licenses for everyone who touches the data, a data model setup, an integration or manual load path, and an admin to maintain it. For a focused targeting workflow, you end up paying platform overhead for what is, functionally, import → rules → filter → export.

What the slice looks like in LiteHCP

StepHow it works
1 · importUpload any CSV, map columns visually, choose per-field imputation for gaps. Profiles upsert by HCP ID, so re-imports are safe.
2 · scoreEvery profile gets a priority score (consent + recency + engagement, weights configurable) and a data-quality confidence score (0–100).
3 · enrichJSON rules assign personas, tags, and compliance flags in priority order — with a tester that runs drafts against live data.
4 · segmentAllowlist-validated SQL-like filters with strict (confidence-gated) and inclusive modes.
5 · exportCSV with minimal fields by default; PII export requires an admin toggle and is audited per export.
total setup — upload one file to any PHP 8 host, register, click Load sample. A 200-profile demo dataset with 8 rules and 5 segments loads instantly so you can evaluate the whole pipeline before importing real data.

When Salesforce is still the answer

  • You already run Salesforce and your HCP engagement lives in journeys, flows, and connected clouds.
  • You need call planning, case management, consent orchestration, or omnichannel execution in one validated platform.
  • Your compliance organization requires a certified vendor stack with formal SLAs.

LiteHCP exports plain CSVs precisely so it can coexist with platforms like Salesforce: build the segment here, execute the campaign wherever you already work.
