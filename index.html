LiteHCP Targeter
Portable single-file PHP + SQLite targeting tool
Overview
Import → enrich → segment → simulate → export

LiteHCP Targeter runs as litehcp.php and stores data in SQLite (WAL mode) for shared hosting. It’s privacy-first by default (exports exclude PII unless enabled by admin).

Run

Open litehcp.php to register the first admin and load the sample dataset.
