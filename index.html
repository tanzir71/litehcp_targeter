Workflow

Import → enrich → segment → simulate → export. All local by default.

CSV import

Upload CSV, map columns, choose imputation strategies, then import in chunks.

Rules enrichment

Apply JSON rules to generate persona, priority score, compliance flag, and tags.

Segments

Build segments from rules or safe SQL-like filters (allowlisted fields only).

Simulator + export

Project contacts/cost/ROI and export privacy-first CSV lists (PII gated by admin).

Stack

Single-file PHP app optimized for shared hosting (offline after setup).
