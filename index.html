Quickstart
1) Upload

Drop litehcp.php (and this index.html) into public_html/.

2) Register

First user becomes admin. Use Dashboard → Load sample to seed data.

3) Operate

Export defaults to minimal fields (no email) unless admin enables PII export.

Workflow
Import → enrich → segment → simulate → export

All local by default. Optimized for small teams and shared hosting.

Import

Upload CSV, map columns, choose imputation strategies, import in chunks.

Rules enrichment

Apply JSON rules to generate persona, priority score, compliance flag, and tags.

Segments

Build segments from rules or safe SQL-like filters (allowlisted fields only).

Simulator + export

Project contacts/cost/ROI and export CSV lists (privacy-first by default).
